# Carousel





















| 属性名  | 说明             |
| ------- | ---------------- |
| height | 高度 |
| initial-index |  初始状态激活的幻灯片索引，从0开始      |
| trigger |  指示器触发的方式      |
| autoplay |  是否自动切换      |
| interval |  自动切换时间间隔，单位毫秒      |
| indicator-position |  指示器的位置      |
| arrow |   切换箭头的显示时机     |
| loop |   是否循环播放     |
| direction |   展示的方向     |
| pause-on-hover |   鼠标悬浮时暂停自动切换     |
