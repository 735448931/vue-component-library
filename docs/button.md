## Api

| 属性名    | 说明             | 类型 |
| --------- | ---------------- | ---- |
| type      | 类型             |      |
| plain     | 是否为朴素按钮   |      |
| round     | 是否为圆角按钮   |      |
| circle    | 是否为圆形按钮   |      |
| loading   | 是否为加载中状态 |      |
| disabled  | 是否为禁用状态   |      |
| autoSpace | 是否自动插入空格 |      |
| icon      | 图标名称         |      |
